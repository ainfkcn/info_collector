# 为AI时代自留的语料清洗和收集库

## zhihu_favorutie

收藏夹使用油猴脚本[知乎收藏夹导出](https://greasyfork.org/zh-CN/scripts/551983-%E7%9F%A5%E4%B9%8E%E6%94%B6%E8%97%8F%E5%A4%B9%E5%AF%BC%E5%87%BA)批量导出收藏夹后，在本地对收藏夹的内容进行清洗；并根据自己的需要打标签。

已实现功能：
- 切分
- 打自定义标签
  - 多收藏夹归一化
  - 原始hash值（防止图床url变更过滤掉图床url）
- 补充元数据
  - 作者名
  - 作者id（防止频繁改名星人）
  - 创作日期
  - 编辑日期
